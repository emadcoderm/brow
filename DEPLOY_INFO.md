## 🌐 Deploy:

### Vercel (پیشنهادی):
1. برو به **vercel.com**
2. Sign in با GitHub
3. "Add New Project"
4. Repository را انتخاب کن
5. Deploy!

### Netlify:
1. برو به **netlify.com**
2. Sign in با GitHub
3. "Add new site" → "Import from Git"
4. Deploy!
